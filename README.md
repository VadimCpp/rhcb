# Getting Started

To set up dev env:

```bash

# Clone the repo
git clone https://github.com/VadimCpp/rhcb.git rhcb

# Go to the project directory
cd rhcb

# Install dependencies
npm install

```

Use commands below:

```bash

# To run development
npm run dev

# To build release for deploying
npm run build

# To deploy website to your github pages
npm run deploy

```
